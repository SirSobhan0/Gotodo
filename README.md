# Gotodo

A simple task and time tracking project in golang

## How to install

### source code
clone the source code:
```bash
git clone https://github.com/SirSobhan0/Gotodo.git
cd Gotodo
go build -o Gotodo .
```

### ||

### go install
```bash
go install github.com/SirSobhan0/Gotodo@latest
```
